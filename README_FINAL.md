## 🚀 快速部署步骤（仅需3步！）

### Step 1: 修改配置 (2分钟)
```toml
# 编辑 hugo.toml
baseURL = 'https://yourusername.github.io/'  # ← 改这里
```

### Step 2: 创建GitHub仓库 (3分钟)
- 仓库名称: `yourusername.github.io`
- 设置为: Public

### Step 3: 推送代码 (5分钟)
```powershell
cd myblog
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/yourusername.github.io.git
git branch -M main
git push -u origin main
```

**或使用快速脚本:**
```powershell
.\deploy.ps1
```

## 📞 问题诊断速查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 网站无法访问 | 仓库名错误 | 确保是 `yourusername.github.io` |

| 文章不显示 | draft设置 | 改为 `draft: false` |
| 构建失败 | 配置错误 | 查看 GitHub Actions 日志 |
